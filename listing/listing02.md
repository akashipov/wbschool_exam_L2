Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)

/// Нужно проработать получше!!!!
func test() (x int) {
	defer func() {
		x++
	}() // func1
	x = 1
	return
}

func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	// fmt.Println(anotherTest())
}
```

Ответ:
```
Будет вывод 2 а затем 1
Это происходит потому что у нас на выходе из функции значение копируется
В методе test переменная х уже ссылается на область памяти для этой самой копии. То есть копирования не происходит
Defer вызывается и значение меняется снаружи от вызова метода

В другом кейсе вызов дефер происходит после копирования return переменной уже наружу
Переменная меняется при вызове дефер. Но это уже старый участок памяти, поэтому снаружи функция main уже не видит этих изменений. Я понимаю это так
```
