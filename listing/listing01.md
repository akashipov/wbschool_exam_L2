Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Выведется [77, 78, 79]
мы возьмем слайс от массива а с первого элемента по последний, последний не включается в результат
```
